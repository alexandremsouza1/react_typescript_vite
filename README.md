## Command
- Install ````npm install ````
- Run dev: ```` npm run dev ````
- Run build develop: ```` npm run build:dev````
- Run build production: ```` npm run build:product ````
- Run preview ````npm run preview ````
- Check lint ````npm run lint ````
- Fix lint ````npm run fix:lint ````
- Format ````npm run format ````
